Installation

WARNING!

Sound support in mozilla is still a work in progress. At this time, mozmp does not work with recent mozilla builds, including M18, 0.7 and 0.8. This release should be upwards compatible with whatever mozilla release sound support works again in.

Step 1

Download and install mozmp by clicking on the button below. On UNIX, this must be done as root.

Step 2

Exit and then re-start mozilla. This will register mozmp with mozilla.

Step 3

Start mozmp. Type:

mozilla -chrome chrome://mozmp/content/
