Installation

WARNING!

Sound support in mozilla is a work in progress. mozmp is only compatible with mozilla 0.8.1.

There is also a problem with some .wav's. Mozilla is not able to play all .wav's. What .wav's will and will not work are still not known. This is something that is still being looked into.

Step 1

Download and install mozmp by clicking on the button below.
On UNIX, this must be done as root.

Step 2

Exit and then re-start mozilla. This will register mozmp with mozilla.
On UNIX, this must be done as root.

Step 3

Start mozmp. Type:

mozilla -chrome chrome://mozmp/content/
